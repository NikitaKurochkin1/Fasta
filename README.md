# FastaReader
Прогрмма создана в качестве домашней работы студента Ивана Алферова и служит для того, что читать файлы формата fastaи немного анализировать их.

## Установка
1. Для использования FastaReader скачайте py файл с таким названием с моего [GitHub](https://github.com/Hellkatze/FastaReader) .

2. Поместите его в той же дериктории, где находится ваш .fasta файл.

3. Откройте в любой программе, поддерживающей выполнение py файлов.

## Использование
- При обычном запуске вам сначала предложит ввести название файла или полный путь до него, если он не в той же папке, что FastaReader.py. После этого вам выведет всю основную информацию о нём.

- Если вам не нужно всё, то в полме «Менять выводные данные здесь:» можно поменять вывод. Обязательно прописать out. и сразу после, без пробела, одну из этих функций:
    - len() — Вычисляет длину последовательности и выводит в консоль результат.

    - str() — Приводит последовательность к строковому виду.

    - seq_alphabet() — Определяет тип последовательности (нуклеотиды или протеины).
